When I made my own Tetris game when I was 13-years-old, I was so excited to show it to my family and friends. Seeing their faces light up while playing my game made me really happy.

Dear My Cat

React Native, Redux, React Router

Mobile simulation game built with React Native. The gameplay is 'Dating with your ideal type of cat'. Users can choose an ideal type of cat and then spend time with the cat until they see their own ending. You can be 'Nyanmates' with your cat, or something bad might happen..?!

SkyLodge

React, CSS Module, Node.js, MongoDB, Redis, Docker, AWS EC2

Web application which allows users to book accommodations. I was responsible for a booking module which gets booking information from users and send it to the express server. Users can send check-in/check-out dates through the calendar and send guest numbers.

QuickStarter

React, SASS, Node.js, PostgreSql, Redis, Docker, AWS EC2/RDS/ELB

Crowdfunding app which allows users to make pledges for prototype products. I was focused more on the backend. I was responsible for its campaign module which shows different price levels of the pledges. Originally the module had 100 entries in Mongo. But I generated 300M entries to conduct stress testing with a realistic scenario and switched the database to Postgres for the relational schema.
